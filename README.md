```json
{
    "prefix": "$",
    "token": "your-bot-token-here",
```

# ⚡ Getting Started

Install Node.js **[Link](https://nodejs.org/en/download/)**

Do These Steps in a Terminal Type Enviroment
- cd Bender
- npm i
- node main.js
